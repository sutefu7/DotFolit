
・このプログラムは何ですか？

	このプログラムは、既存の Windows フォームアプリケーションのプログラム理解のための、サポートアプリという位置づけで、企画・作成しました。
	具体的には、担当を任された時に、（対応する仕様書はすでに理解しているとして）ソースコードを最短で理解しなければならない場合などで、利用します。

	実際に使ってみて、使えるのか使えないかは、なんだかわからなくなってしまいました。
	理解の仕方や流れは人それぞれだと思いますし、このアプリがあなたにとって最適だった場合は、あなたはラッキーです。

	（容量が大きいため）exe 形式では配布しません。
	Visual Studio 2015 以上の環境でビルドして作成して、使ってください。



・開発環境

	本プログラムは、以下の環境で作成しています。
	Windows 8.1 Pro
	Visual Studio Community 2015
	VB.NET, WPF アプリケーション(.NET Framework 4.6)
	
	Visual Studio 標準コントロールでしか確認していないので、
	外部ベンダー提供のコントロールを使ったアプリケーションでもうまく動作するのかは未確認です。
	また、.NET Framework 3.5 以下の Windows フォームアプリケーションを解析できるかどうかも未確認です。
	(例えば、Windows 7 & Visual Studio 2005(.NET Framework 2.0) で作成したものを解析できるかどうかという意味の方で)

	TreeListView コントロールのこと
	TreeView のまま使うか、TreeListView に差し替えて使うか迷ったが、結局 TreeView のままにした
	TreeListView だと、第２項目が綺麗に見える

	(TreeView の場合)
	Property
	  + Width : Integer
	  + ActualWidth : Integer
	  
	(TreeListView の場合)
	メンバー名          型名
	---------------------------
	Property
	  + Width         Integer
	  + ActualWidth   Integer


	ただし、長いメンバー名を表示しようとして、列を広げてしまうと、短いメンバー名が見づらくなるかと思い、採用は止めた
	（見づらかったら、列を縮めてください。で済む話ではあるが）
